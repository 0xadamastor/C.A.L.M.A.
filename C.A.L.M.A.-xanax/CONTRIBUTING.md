# Contributing to CALMA / Contribuir para o CALMA

## English

### How to Contribute

1. **Fork** the repository
2. Create a **feature branch**: `git checkout -b feature/my-feature`
3. **Commit** your changes: `git commit -m 'Add new feature'`
4. **Push** to the branch: `git push origin feature/my-feature`
5. Open a **Pull Request**

### Code Guidelines

- Follow PEP 8 for Python code
- Keep functions small and focused
- Add docstrings to new functions
- Test your changes before submitting

### Reporting Issues

- Use the GitHub Issues page
- Include steps to reproduce the problem
- Attach relevant log excerpts if applicable

## Portugues

### Como Contribuir

1. Faz **Fork** do repositorio
2. Cria um **branch**: `git checkout -b feature/minha-funcionalidade`
3. Faz **Commit** das alteracoes: `git commit -m 'Adiciona nova funcionalidade'`
4. Faz **Push** do branch: `git push origin feature/minha-funcionalidade`
5. Abre um **Pull Request**

### Guia de Codigo

- Segue PEP 8 para codigo Python
- Mantem funcoes pequenas e focadas
- Adiciona docstrings a novas funcoes
- Testa as alteracoes antes de submeter

### Reportar Problemas

- Usa a pagina de Issues do GitHub
- Inclui passos para reproduzir o problema
- Anexa excertos de logs se aplicavel
